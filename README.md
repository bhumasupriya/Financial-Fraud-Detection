# 💳 Real-Time Financial Fraud Detection System

## 📖 Project Overview
The **Real-Time Financial Fraud Detection System** uses Machine Learning and Data Analytics to identify suspicious financial transactions and detect fraudulent activities.  

This system provides **real-time fraud monitoring** through a predictive machine learning model and an interactive dashboard. It can help banks, fintech companies, and financial institutions minimize financial risks and prevent fraudulent transactions.

The project integrates **supervised learning, unsupervised anomaly detection, and real-time data processing** to detect fraud patterns effectively.

---

## 🎯 Objectives
- Detect fraudulent transactions using machine learning models
- Provide real-time monitoring of transactions
- Visualize fraud statistics through an interactive dashboard
- Identify suspicious transaction patterns and anomalies

---

## 🧠 Machine Learning Models Used

### Supervised Learning
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- XGBoost
- LightGBM

### Unsupervised Learning
- Isolation Forest
- One-Class SVM

### Deep Learning
- Autoencoder for anomaly detection

### Graph-Based Analysis
- Transaction network analysis to detect fraudulent relationships.

## 🚀 How to Run the Project

1️⃣ Install Dependencies

pip install pandas numpy scikit-learn flask streamlit plotly xgboost lightgbm
2️⃣ Train the Model
python train_model.py
3️⃣ Start Fraud Detection API
python app.py
4️⃣ Start Real-Time Transaction Stream
python stream_data.py
5️⃣ Launch Dashboard
streamlit run dashboard.py
